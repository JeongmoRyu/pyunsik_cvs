## 📌 프로젝트 소개

### ✔️ 주제

건강한 편의점 조합 추천 앱

### ✔️ 주요 기능

1. 영양 성분 정보 제공
2. 나만의 조합 생성
3. 행사 상품 목록 제공
4. 추천 서비스
    1. 유저 기반 상품 추천
    2. 영양정보 기반 상품 추천
    3. 조합 기반 상품 추천
5. 실시간 인기 검색어 제공

### ✔️ 개발 기간

2023.08.28 - 2023.10.06
